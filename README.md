## Installation

- Clone this repository:

    ```
    git clone https://github.com/m1neral/ororo-search-app.git
    ```
- Install gems:

    ```
    bundle
    ```
- Setup database:

    ```
    rails db:setup
    ```
- Install frontend environment:
    ```
    cd frontend
    npm i
    npm run dev
    ```
- Start the server:
    ```
    rails s
    ```
